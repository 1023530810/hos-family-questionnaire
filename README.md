## 怎么填（约 5–10 分钟）

1. 打开链接（手机可填）
2. 从上到下填 **A→G**（单页，不用换页签）
3. 点 **「保存并提交云端」**
4. 把生成的云端链接发回对话（或导出 JSON 备份）
